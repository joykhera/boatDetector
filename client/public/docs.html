GraphQL Boat Queries Documentation

getBoat

Description: Fetch a specific boat by its ID.

Arguments:

  • id (GraphQLID): The ID of the boat to be retrieved.
Return Type: BoatType

Example:

    {
        getBoat(id: 1) {
            id
            link
            size
            speed
            timestamp
        }
    }

getBoats

Description: Fetch all boats from the database.

Arguments: None
Return Type: Array of BoatType

Example:

    {
        getBoats {
            id
            link
            size
            speed
            timestamp
        }
    }
